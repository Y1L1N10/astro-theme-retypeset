## 📱 导航与菜单类

| 英文术语 | 中文译名 | 说明 |
|---------|---------|------|
| Navigation Bar | 导航栏 | 顶部显示标题和返回按钮的区域 |
| Tab Bar | 标签栏 / 底部导航栏 | 底部切换主要功能的导航栏（移动端常见）|
| Menu | 菜单 | 功能选项列表 |
| Menu Bar | 菜单栏 | 水平或垂直排列的菜单选项（桌面端常见）|
| Navigation Menu | 导航菜单 | 包含导航链接的菜单组件 |
| Hamburger Menu | 汉堡菜单 | 三条横线的侧边菜单图标（≡）|
| Overflow Menu / Kebab Menu | 溢出菜单 / 更多菜单 | 竖排三个点的更多选项按钮（⋮）|
| Meatballs Menu | 肉丸菜单 | 横排三个点的更多选项按钮（⋯）|
| Bento Menu | 便当菜单 / 网格菜单 | 网格状排列的菜单选项 |
| Sidebar | 侧边栏 | 从左或右滑出的菜单面板 |
| Drawer | 抽屉菜单 | 从屏幕边缘滑出的导航面板 |
| Toolbar | 工具栏 | 显示操作按钮的横条 |
| Breadcrumb | 面包屑导航 | 显示当前页面路径的导航链接 |

## 🖼️ 内容展示类

| 英文术语 | 中文译名 | 说明 |
|---------|---------|------|
| Status Bar | 状态栏 | 最顶部显示时间、电量等系统信息 |
| Header | 页头 / 头部 | 页面顶部区域 |
| Footer | 页脚 / 底部 | 页面底部区域 |
| Card | 卡片 | 包含相关信息的独立内容块 |
| List | 列表 | 纵向排列的内容项 |
| Feed | 信息流 / 动态流 | 持续更新的内容流（如朋友圈、新闻流）|
| Banner | 横幅 / Banner | 顶部或页面中的广告/通知条 |
| Carousel | 轮播图 / 走马灯 | 可滑动浏览的图片或内容循环展示 |
| Accordion | 手风琴 / 折叠面板 | 可展开/折叠的内容区域 |
| Hero Section | 首屏大图区 | 网页顶部的大型视觉区域 |
| Widget | 小部件 / 组件 | 独立的功能模块或信息展示单元 |
| Thumbnail | 缩略图 | 小尺寸预览图 |
| Gallery | 图库 / 画廊 | 图片集合展示区 |

## 🔘 交互元素类

| 英文术语 | 中文译名 | 说明 |
|---------|---------|------|
| Button | 按钮 | 可点击的操作按钮 |
| FAB (Floating Action Button) | 悬浮按钮 | 圆形悬浮的主操作按钮（Material Design）|
| Toggle / Switch | 开关 / 切换器 | 二选一的切换控件 |
| Slider | 滑块 / 滑动条 | 调节数值的滑动控件 |
| Input Field / Text Field | 输入框 / 文本框 | 文本输入区域 |
| Text Area | 文本域 / 多行文本框 | 多行文本输入区域 |
| Dropdown / Select | 下拉菜单 / 选择器 | 展开选项的选择控件 |
| Checkbox | 复选框 | 多选框 |
| Radio Button | 单选按钮 | 单选框 |
| Knob | 旋钮 | 旋转调节的控件（常用于音频应用）|
| Date Picker | 日期选择器 | 选择日期的控件 |
| Time Picker | 时间选择器 | 选择时间的控件 |
| Color Picker | 颜色选择器 | 选择颜色的控件 |
| Stepper | 步进器 | 增减数值的控件（+ - 按钮）|
| Range Control | 范围控制器 | 选择数值范围的控件 |

## 💬 提示与反馈类

| 英文术语 | 中文译名 | 说明 |
|---------|---------|------|
| Toast | 轻提示 / 吐司提示 | 短暂显示的小提示（自动消失）|
| Snackbar | 操作提示条 | 底部弹出的可操作提示（Material Design）|
| Alert | 警告框 / 提醒 | 需要用户确认的弹窗 |
| Dialog | 对话框 | 模态弹窗，需要用户交互 |
| Modal | 模态窗口 | 覆盖全屏的弹出层 |
| Popup | 弹出窗口 | 小型弹出窗口 |
| Tooltip | 工具提示 / 悬浮提示 | 鼠标悬停或长按显示的说明 |
| Badge | 徽章 / 角标 | 显示数字或状态的小圆点（如未读消息数）|
| Callout | 标注 / 提示框 | 指向性的说明框（带箭头）|
| Notification | 通知 | 系统或应用推送的消息提醒 |
| Progress Bar | 进度条 | 显示任务进度的横条 |
| Loading Indicator / Spinner | 加载指示器 / 加载动画 | 显示正在加载的动画效果 |
| Skeleton Screen | 骨架屏 | 内容加载前的占位动画 |
| Confirmation Dialog | 确认对话框 | 要求用户确认操作的弹窗 |

| 英文术语 | 中文译名 | 说明 |
|---------|---------|------|
| H5 Page | H5 页面 | 基于 HTML5 的移动网页 |

| Hybrid App | 混合应用 | 结合原生和 Web 技术的应用 |

| Bottom Sheet | 底部弹出层 | 从底部滑出的面板 |

| 英文术语 | 中文译名 | 说明 |
